如何使用 作品快傳盒 系統

1 申請與配置您的個人 Firebase

作品快傳盒採用「無伺服器託管」架構，您的圖片與資料都會直接存放在您專屬的免費 Firebase 空間中，100% 保障學生的隱私安全！請按照以下步驟完成設定：

  1. 登入主控台： 前往 Firebase Console 官網，並使用您的 Google 帳號進行登入。
  2. 建立新專案 (Project)： 點擊畫面的 新增專案 (Add Project) ➜ 輸入專案名稱（例如：quick-uploader）➜ 依序點選繼續（推薦關閉 Google Analytics 較快完成）➜ 完成建立。
  3. 建立 Firestore 資料庫： 在專案控制面板的左側選單點擊 Firestore Database ➜ 點擊 建立資料庫 (Create database) ➜ 選擇伺服器位置（可選 asia-east1 台灣）➜ 安全設定務必選擇 以測試模式啟動 (Start in test mode) ➜ 點擊啟用。
  4. 註冊網頁應用程式 (Web App)： 點擊控制面板左上角「專案總覽 (Project Overview)」旁的 齒輪 ➜ 專案設定 ➜ 拉到下方「您的應用程式」區塊 ➜ 點擊網頁圖示 (Web App) ➜ 輸入暱稱（例如：artbox-web）➜ 點擊註冊。
  5. 複製設定 Config 物件： 註冊成功後，畫面會顯示包含 const firebaseConfig = { ... }; 的程式碼。請複製其中括號內的六行內容（也就是 apiKey 到 appId）。
  6. 設定並連線： 複製後，回到本平台網頁，點擊右上角的 設定按鈕，將內容貼到設定輸入框中並點選儲存。網頁將會重新整理並建立專屬您的雲端快傳資料庫！

2 發布任務與展示

教師透過電腦開啟本系統，輸入預設密碼「tpet」進入教師入口。點選發布新任務並將學生 QR Code 投放到大螢幕上。學生即可使用平板拍照隨時同步傳送！

3 推薦：Firestore 安全規則設定

請在您的 Firebase Firestore 規則中貼上下列程式碼，即可保證系統運作：

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /artifacts/{appId}/public/data/{collection}/{document} {
      allow read, write: if true;
    }
  }
}
